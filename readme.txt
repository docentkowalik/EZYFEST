EZYFEST is a content management system targeted at festivals. This CMS can be utilised by the festivals to create and manage their website containing all the most useful information that should be found on every festival website. The Dashboard and the Front End dynamically generated web pages are fully mobile optimised allowing festival administrators to create and update their web content on a mobile device but also allowing users to view websites on their mobile devices.

EZYFEST content management system provides all the essential tools required to design and publish a festival website without any programming knowledge. EZYFEST shortens the development time to a minimum and allows for very efficient web development.

Technologies Used: Node.js, Express.js, MongoDB, EJS - Javascript Templates


Installation:      
	1. npm install
	2. node app
